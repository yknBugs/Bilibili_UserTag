# Bilibili_UserTag
根据动态中的关键词，给b站评论区用户加标签的脚本

## 说明
该js脚本参考原作者@Xulaupuz 脚本:原神玩家指示器 改编

相比于原版本，大幅修改了检测关键词的定义方式，使得自定义增加更多的标签更容易

可以使用仓库中的cpp文件，将*关键词列表*转化为*关键词设置(js代码)*
创建一个名称为text.txt的文件，然后将要转化的关键词输入进该文件中。每个关键词占一行，行末不能有空格，不能有多余的换行。随后编译运行KeywordCodeGenerate.cpp文件，即可生成一个output.txt文件，里面就是生成的js代码片段，也就是js中用来设置关键词的代码片段（关键词颜色为随机生成）。将这个代码片段复制到脚本的.js文件中的对应位置即可。
